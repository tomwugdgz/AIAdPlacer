## 二、三只虾功能定义

### 1. 选题虾 (TopicBot)
**功能**：根据客户行业、产品、目标人群，生成广告选题和创意方向

**输入**：
- `industry`：行业（日化/药店/快消/餐饮…）
- `product`：产品名称
- `target_audience`：目标人群
- `campaign_goal`：营销目标（品牌曝光/效果转化/活动推广）

**输出**：
```json
{
  "topics": [
    {
      "title": "选题标题",
      "angle": "切入角度",
      "key_message": "核心信息",
      "scenario": "适用场景"
    }
  ]
}
```

**Prompt 模板**：
```
你是一位资深广告策划。请根据以下信息生成 3 个广告选题：

行业：{industry}
产品：{product}
目标人群：{target_audience}
营销目标：{campaign_goal}

要求：
1. 每个选题包含：标题、切入角度、核心信息、适用场景
2. 突出社区媒体特性（家庭场景、高频触达）
3. 符合品牌调性

输出 JSON 格式。
```

---

### 2. 文案虾 (CopyBot)
**功能**：根据选题生成多版本广告文案，支持 A/B Test

**输入**：
- `topic`：选题（来自选题虾）
- `style`：文案风格（专业/接地气/幽默/温情）
- `length`：文案长度（短/中/长）
- `platform`：投放平台（开门广告/灯箱/App开屏）

**输出**：
```json
{
  "copies": [
    {
      "version": "A版",
      "content": "文案内容",
      "hook": "吸引点",
      "cta": "行动号召"
    }
  ]
}
```

**Prompt 模板**：
```
你是一位资深文案。请根据以下选题写广告文案：

选题：{topic}
风格：{style}
长度：{length}
平台：{platform}

要求：
1. 开门见山，3 秒内抓住注意力
2. 突出产品利益点
3. 有明确的行动号召（扫码/到店/下单）
4. 社区场景化表达

输出 JSON 格式，包含 2 个版本（A/B）。
```

---

### 3. 审核虾 (ReviewBot)
**功能**：审核文案质量，输出评分和修改建议

**输入**：
- `copy`：待审核文案
- `brand_guidelines`：品牌规范（可选）
- `compliance_rules`：合规要求（广告法/平台规则）

**输出**：
```json
{
  "score": 85,
  "passed": true,
  "issues": [],
  "suggestions": ["建议1", "建议2"],
  "optimized_copy": "优化后的文案"
}
```

**Prompt 模板**：
```
你是一位广告合规审核专家。请审核以下文案：

文案：{copy}
品牌规范：{brand_guidelines}
合规要求：{compliance_rules}

审核维度：
1. 合规性（是否违反广告法）
2. 品牌一致性（是否符合品牌调性）
3. 吸引力（是否能抓住注意力）
4. 清晰度（信息是否明确）
5. 行动号召（是否有明确的 CTA）

输出 JSON 格式，包含：总分（0-100）、是否通过、问题列表、修改建议、优化后文案。
```

### 2. 核心服务实现

#### `topic_bot.py` - 选题虾
```python
from app.bmn.llm_client import call_llm
import json

def generate_topics(industry, product, target_audience, campaign_goal, num_topics=3):
    """生成广告选题"""
    prompt = f"""你是一位资深广告策划。请根据以下信息生成 {num_topics} 个广告选题：

行业：{industry}
产品：{product}
目标人群：{target_audience}
营销目标：{campaign_goal}

要求：
1. 每个选题包含：标题、切入角度、核心信息、适用场景
2. 突出社区媒体特性（家庭场景、高频触达）
3. 符合品牌调性

输出 JSON 格式：
{{
  "topics": [
    {{
      "title": "...",
      "angle": "...",
      "key_message": "...",
      "scenario": "..."
    }}
  ]
}}"""

    response = call_llm(prompt, timeout=60)
    
    # 解析 JSON（处理可能的 markdown 代码块）
    if "```json" in response:
        response = response.split("```json")[1].split("```")[0].strip()
    elif "```" in response:
        response = response.split("```")[1].split("```")[0].strip()
    
    return json.loads(response)
```

### 3. 编排服务 (`content_service.py`)

```python
from app.bmn.workflows.topic_bot import generate_topics
from app.bmn.workflows.copy_bot import generate_copies
from app.bmn.workflows.review_bot import review_copy

def generate_ad_content(industry, product, target_audience, campaign_goal, style="专业"):
    """
    完整的内容生产流程：选题 → 文案 → 审核
    
    :return: {
        "topics": [...],
        "copies": [...],
        "reviews": [...]
    }
    """
    # 1. 选题
    topics_result = generate_topics(industry, product, target_audience, campaign_goal)
    topics = topics_result.get("topics", [])
    
    # 2. 文案（取第一个选题）
    if topics:
        topic = topics[0]["title"] + "：" + topics[0]["key_message"]
        copies_result = generate_copies(topic, style=style)
        copies = copies_result.get("copies", [])
    else:
        copies = []
    
    # 3. 审核（取第一个文案）
    reviews = []
    for copy in copies:
        review_result = review_copy(copy["content"])
        reviews.append(review_result)
    
    return {
        "topics": topics,
        "copies": copies,
        "reviews": reviews
    }
```
