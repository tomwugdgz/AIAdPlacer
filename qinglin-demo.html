四大智能引擎

基于 LangGraph 编排的四个专业 AI Agent，覆盖从人群洞察到效果归因的完整投放链路。

Smart Reply 智能回复

AI 人群洞察 Agent 通过 KMeans 聚类与 DMP 标签分析，自动识别高价值受众群体，输出精准人群包。支持自然语言交互，一句话完成复杂定向。

📄

Document Analysis 文档分析

内置 RAG 向量检索引擎，基于 ChromaDB 存储行业知识与投放策略。快速解析 PDF 报告、竞品数据，自动生成投放建议方案。

Code Review 代码审查

AI 智能排期 Agent 运用 FM + deepMCP 多目标优化算法，在预算约束下自动产出最优排期方案。内置合规检测，确保投放策略安全可控。

📊

Team Insights 团队洞察

效果归因 Agent 打通线上线下数据，基于真实到店轨迹构建可信归因模型。生成多维度可视化报告，驱动持续优化迭代。

核心能力

五大核心数据定向能力

打破传统户外广告难以精准、无法联动的局限，让每一块屏幕成为智能沟通节点。

曝光量统计

依托 MAM 系统地铁曝光量测算模型，结合车站人数、客流线、媒体尺寸、广告播放时长，算出核心指标曝光量，实现曝光效率与成本的精准平衡。

POI 场景定向

借助移动设备定位数据及第三方 LBS 服务，基于受众与特定兴趣点的邻近关系，实现超本地化触达与高度适配的信息传递。

TA 标签定向

整合第三方数据服务商、品牌自有 CRM 系统等多渠道数据，提炼受众人口统计、消费心理、行为偏好等核心标签，精准定义目标人群画像。

外部数据联动定向

联动天气、实时位置、消费动态等外部数据，将 pDOOH 投放与现实环境因素深度绑定。基于数据触发动态调整广告创意内容，显著提升广告相关性。

跨渠道再营销定向

将 pDOOH 曝光过的受众整合为专属人群包，实现营销资产的二次利用。对目标受众开展跨渠道、多触点的持续曝光，提升整体营销链路转化效率。
