2.1 核心术语定义

术语英文定义来源

2.2 核心计算公式（共6条）

公式1: 流动曝光量（IMPflow）

TCAAAD §7.3 公式(1) IMPflow = min(OTCflow × SOT, 1) × Traffic
流动人群造成的广告曝光量 = 曝光概率 × 时间占比 × 人群流量，上限为单次曝光

公式2: 流动曝光概率（OTCflow）

TCAAAD §7.3 公式(2) OTCflow = (max(Texposure − 2, 0) + Tad) / Tad
Texposure: 曝光时长（秒）| Tad: 广告时长（秒）
规则: 观看不足1秒不计有效曝光，理论值需满足至少观看2次

公式3: 驻留曝光量（IMPdwell）

TCAAAD §7.3 公式(3) IMPdwell = OTCdwell × Traffic
驻留人群造成的曝光量 = 驻留曝光概率 × 人群流量

公式4: 驻留曝光概率（OTCdwell）

TCAAAD §7.3 公式(4) OTCdwell = Texposure ÷ 300
每额外停留5分钟（300秒），增加一个观看次数。源于全球户外广告行业通用规则

公式5: 总曝光量（IMP）

TCAAAD §7.3 公式(5) IMP = IMPflow + IMPdwell
单个广告点位总曝光量 = 流动曝光 + 驻留曝光

公式6: 曝光乘数（Impression Multiplier）

TCAAAD §7.4 公式(6) Impression Multiplier = Traffic × IMPflow ÷ ad_slots
衡量每播放一次广告片所产生的曝光量。ad_slots = 广告位槽位数

2.3 迪杰斯特拉算法（最短路径）

🔍 算法应用
在地铁/机场场域中，使用迪杰斯特拉算法（Dijkstra's algorithm, 1956）计算人群中各进出节点间的最短路径。将场域内人群流动建模为有向带权图，节点间权重之和 = 1。结合OD矩阵（起点/终点矩阵）与交通数据得出人群流动总量。

7.3 关键差异与互补性

TCAAAD vs 受众测量

标准 vs 实现

TCAAAD定义了标准公式框架，秒针受众测量是OTC模型的实际应用。两者公式逻辑一致（都基于曝光时长、人群流量、概率计算）。

BrandLift vs 屈臣氏白皮书

测量 vs 运营

BrandLift专注广告对品牌KPI的测量（认知/偏好/购买倾向），屈臣氏白皮书将品牌KPI映射为AECR人群资产进行运营。

跨屏测量能力

唯一方案

秒针受众测量方案提供了MixReach跨屏Reach评估能力（ID去重/系统模型推演），四份文档中唯一涵盖线上线下整合测量的方案。
