## 🆕 新增功能

### 1. ROI Agent 服务 (`backend/app/roi_agent.py`)

**端口**: 5004（独立服务）/ 内嵌路由 `/api/v2/roi/*`

**核心能力**:
- 📈 **三场景 ROI 计算**（`/calculate` + `/three-scenarios`）：悲观/中性/乐观三档预设
- 📊 **灵敏度分析**（`/sensitivity`）：单参数变动对 ROI 影响曲线
- 📋 **多方案对比**（`/compare`）：并排对比多个投放方案
- 📖 **公式说明**（`/formula`）：完整公式列表 + 参数定义 + 数据来源
- 📉 **行业基准**（`/benchmark`）：社区单元门 vs 梯媒/地铁/电视/线上广告 ROI 横评

**ROI 计算公式（归因自 duckwolf.cn/10.html）**:

| 公式 | 表达式 | 说明 |
|------|----------|------|
| UV（独立触达人数） | `UV = N × U × P × β` | N=框数, U=户数/栋, P=人数/户, β=接触率 |
| PV（总曝光次数） | `PV = UV × γ × T` | γ=日接触频次, T=投放天数 |
| 记忆人数 | `记忆人数 = UV × r` | r=记忆率（0.15/0.18/0.22） |
| 转化人数 | `转化人数 = 记忆人数 × c` | c=转化率（默认 2%） |
| 首期销售 | `首期销售 = 转化人数 × a` | a=客单价（¥20/22/25） |
| LTV（生命周期价值） | `LTV = 首期销售 × (1 + f)` | f=复购系数（1.3/1.4/1.5） |
| ROI（投资回报率） | `ROI = (LTV - 成本) / 成本 × 100%` | 返回百分比 |

**三场景预设**:

| 场景 | 记忆率(r) | 客单价(a) | 复购系数(f) | 8周 LTV ROI |
|------|------------|-------------|----------------|--------------|
| 悲观 | 0.15 | ¥20 | 1.3 | 859% |
| 中性 | 0.18 | ¥22 | 1.4 | 1,367% |
| 乐观 | 0.22 | ¥25 | 1.5 | 2,312% |

**参数数据来源**:
- `[MCP 数据库]`: 711 广州楼盘中位数（U=100 户/栋）
- `[国家统计局]`: 《中国统计年鉴2024》（P=2.51 人/户）
- `[皓邻实测]`: 5 年接触率数据（β=0.85）
- `[CTR 报告]`: 户外广告记忆率行业区间（8-25%）
- `[凯度 Kantar 2024]`: 家庭场景口碑传播（1.2-1.3）

**API 端点**:

| 端点 | 方法 | 功能 |
|--------|------|------|
| `/api/v2/roi/health` | GET | 健康检查 |
| `/api/v2/roi/calculate` | POST | ROI 计算（支持三场景预设） |
| `/api/v2/roi/three-scenarios` | GET | 快速获取三场景 ROI |
| `/api/v2/roi/sensitivity` | POST | 灵敏度分析 |
| `/api/v2/roi/compare` | POST | 多方案对比 |
| `/api/v2/roi/formula` | GET | 公式说明与参数定义 |
| `/api/v2/roi/benchmark` | GET | 行业 ROI 基准对比 |

## 🔧 改进优化

1. **公式透明化**：
   - 每个计算步骤返回完整公式展开（如 `UV = 5000 × 100 × 2.51 × 0.85 = 1,067,000`）
   - 参数来源标注（`[MCP 数据库]` / `[国家统计局]` / `[皓邻实测]`）

2. **三场景快速查询**：
   - `GET /three-scenarios?N=5000&cost=100000` 一行获取三档 ROI
   - 无需构造完整请求体

3. **灵敏度分析**：
   - 支持单参数扫描（如 `variable=r, values=[0.15,0.16,...,0.22]`）
   - 返回 ROI/LTV 变化曲线数据点

4. **行业基准对比**：
   - 内置 6 种媒体 ROI 横评数据
   - 社区单元门 LTV ROI 1,367% 与线上效果广告（800-1,200%）同属第一梯队

## 📦 部署说明

### 启动方式

**方式一：独立服务（端口 5004）**
```bash
cd D:/Mirofish/AIAdPlacer/backend
python run_roi_agent.py
```

**方式二：内嵌到主服务（端口 5002）**
```bash
cd D:/Mirofish/AIAdPlacer/backend
python run.py   # 已包含 ROI Agent 路由
```

### 健康检查
```bash
curl http://127.0.0.1:5004/health
# 或（内嵌模式）
curl http://127.0.0.1:5002/api/v2/roi/health
```

### 测试三场景 ROI
```bash
curl "http://127.0.0.1:5002/api/v2/roi/three-scenarios?N=5000&cost=100000& T=14"
```

### 测试完整计算
```bash
curl -X POST http://127.0.0.1:5002/api/v2/roi/calculate \
  -H "Content-Type: application/json" \
  -d '{
    "N": 5000,
    "U": 100,
    "P": 2.51,
    "beta": 0.85,
    "gamma": 2.0,
    "T": 14,
    "r": 0.18,
    "a": 22,
    "f": 1.4,
    "cost": 100000,
    "scenario": "neutral"
  }'
```
